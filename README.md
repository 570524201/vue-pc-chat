# vue-chat

野火IM VUE Demo，演示如何将野火IM的能力集成到VUE项目。

## 重要提示
项目在活跃开发中，更完整项目，请参考 [web-chat](https://github.com/wildfirechat/web-chat) 。

## 开发
1. ```npm install```
2. ```npm run serve```
